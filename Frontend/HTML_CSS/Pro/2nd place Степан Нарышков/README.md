RUN:
 - install [nodejs](https://nodejs.org/uk/download/)
 - install [gulp](https://gulpjs.com/)
 - install dependencies `npm install`
 - run project `npm run start`

BUILD
 - install [nodejs](https://nodejs.org/uk/download/)
 - install [gulp](https://gulpjs.com/)
 - install dependencies `npm install`
 - build project `npm run build`
